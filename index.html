SpankBang Plugin

Version: 7

Author: GreyPluginDev

Basic Spankbang Plugin for Grayjay the app

Installation

To install this plugin in Grayjay:

  1. Open Grayjay app
  2. Go to Sources → +
  3. Scan the QR code below

Plugin Files

Plugin Configuration (SpankbangConfig.json) Plugin Script (SpankbangScript.js) Plugin Icon

Features Working

  • Video download (HLS)
  • Video search
  • Video playback
  • Video info
  • View profiles for models, pornstars, and channels
  • Channel info fetch (description, about, its videos)
  • Channel subscribe/unsubscribe
  • Profile Search

xHamster Plugin

Version: 1

Author: GreyPluginDev

xHamster video platform plugin for Grayjay

Installation

To install this plugin in Grayjay:

  1. Open Grayjay app
  2. Go to Sources → +
  3. Enter the plugin config URL manually or scan QR code when available

Plugin Files

Plugin Configuration (XhamsterConfig.json) Plugin Script (XhamsterScript.js) Plugin Icon

Features

  • Video playback and download
  • Video search
  • Home feed
  • Channel/User/Pornstar support
  • Related videos
  • Channel subscribe/unsubscribe
